# Hotspot Helper

Bash script for logging in and logging out into captive portals (hotspots).

> Currently only MikroTik RouterOS (UserManager package) is supported

## Usage:
First add executable permission:

    chmod +x hotspot.sh

**In order to log in:**

    ./hotspot.sh USERNAME

**To log out:**

    ./hotspot.sh --logout
